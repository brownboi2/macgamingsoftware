SteamCMD

Description: A CLI for extracting Steam installers. You can install this with the home-brew command: brew install steamcmd or run these commands: mkdir ~/Steam && cd ~/Steam curl -sqL "https://steamcdn-a.akamaihd.net/client/installer/steamcmd_osx.tar.gz" | tar zxvf -
